# blog_project
협업 프로젝트전, 만든 개인 학습용 프로젝트입니다.<br>
간단한 게시판과 홈쇼핑에 필요한 API 설계했습니다.<br>
-학습내용들을 추가하는 프로젝트로, 완성의 끝은 없습니다.

# 사용 스킬
1. SpringBoot
2. Spring Security, OAuth2.0
3. MySQL
4. JPA, Spring Data JPA
5. Thymeleaf, js-ajax


# 기능
1. 회원가입, OAuth2.0
2. 로그인

# API
1. 게시판 글, 댓글 CRUD
2. 게시글 조회
3. 아이템, 주문 CRUD
4. 주문목록 조회
